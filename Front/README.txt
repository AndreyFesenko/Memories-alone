
# Memories IdentityService Frontend (React + Vite + Tailwind, Русский, Dark/Light Theme)

## Быстрый старт

1. Установи Node.js LTS: https://nodejs.org/

2. Распакуй архив, перейди в папку и установи зависимости:

    Открой терминал или PowerShell и перейди в эту папку:
	
    npm install

3. Запусти dev-server:

    npm run dev

4. Открой http://localhost:5035

---
## Переменные окружения

- Чтобы указать URL твоего API, создай `.env`:

    VITE_API_URL=http://localhost:5035

---
## Возможности

- Регистрация и вход
- Просмотр и обновление токенов
- Просмотр своих claims и ролей
- Переключение темы (тёмная/светлая)
- Полная русификация UI
- Минималистичный UI на TailwindCSS
